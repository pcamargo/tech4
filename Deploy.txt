Passo 4: Fazer o Deploy com o GitHub
1. Crie um Repositório no GitHub: Se você ainda não fez isso, crie um novo repositório no GitHub.
2. Envie seu Projeto: Adicione todos os arquivos do seu projeto (Home.py, requirements.txt, as pastas pages, src, files) para o repositório e faça o push.
3. Acesse o Streamlit Cloud:
◦ Vá para share.streamlit.io.
◦ Clique em "New app".
◦ Selecione o repositório e o branch que você acabou de atualizar.
4. Configure o Deploy:
◦ Repository: Seu repositório (ex: seu-usuario/tech4).
◦ Branch: main (ou master).
◦ Main file path: Home.py. Este é o ponto mais importante! Você deve apontar para o seu novo script principal.
◦ Clique em "Deploy!".
O Streamlit irá ler seu requirements.txt, instalar as dependências e iniciar seu aplicativo.
Em alguns minutos, seu dashboard de múltiplas páginas estará no ar, com uma navegação lateral para alternar entre a consulta e o painel analítico.
